🧠 Customer Churn Prediction using ANN

## 📌 Overview
This project predicts customer churn using an Artificial Neural Network.

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- Pandas, NumPy

## 📊 Model
- ANN with Dense layers
- Activation: ReLU, Sigmoid

## 🎯 Result
Achieved ~85% accuracy
